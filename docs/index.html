🔑 Authentication 认证

所有 API 请求需要 Bearer Token。通过以下接口获取 token：

POST /v1/auth/token 获取访问令牌 / Obtain access token

Request Body

curl -X POST https://api.threatledger.net/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{
    "api_key": "tl_live_abc123...",
    "api_secret": "sk_..."
  }'

Response

{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "Bearer",
  "expires_in": 3600,
  "scope": "read write"
}

💡 将 access_token 添加到后续请求头：Authorization: Bearer <token>

⚡ Rate Limits 速率限制

请求速率按套餐等级分配，超出限制将返回 429 错误。

Plan 套餐Requests / minDaily LimitBatch Size
Free1050010
Starter605,00050
Pro300Unlimited100

Response Headers 响应头

每个响应都包含以下速率限制和额度信息头：

HeaderDescription
X-RateLimit-Limit当前窗口允许的最大请求数
X-RateLimit-Remaining当前窗口剩余请求数
X-RateLimit-Reset窗口重置时间（Unix timestamp）
X-Credits-Remaining账户剩余额度
X-Credits-Consumed本次请求消耗的额度

🚨 Error Codes 错误码

API 使用标准 HTTP 状态码，错误响应格式如下：

{
  "error": {
    "code": 401,
    "type": "unauthorized",
    "message": "Invalid or expired access token"
  }
}
CodeTypeDescription 描述
400Bad Request请求参数错误或缺失 — Missing or invalid parameters
401Unauthorized认证失败，Token 无效或过期 — Invalid/expired token
404Not Found请求的资源不存在 — Resource not found
429Too Many Requests超出速率限制，请稍后重试 — Rate limit exceeded

💻 Code Examples 代码示例

常用语言的集成示例 / Integration examples in popular languages

JavaScript (fetch)

const checkAddress = async (address) => {
  const res = await fetch(
    `https://api.threatledger.net/v1/addresses/check?address=${address}`,
    {
      headers: {
        "Authorization": `Bearer ${API_TOKEN}`,
      },
    }
  );

  if (!res.ok) {
    const err = await res.json();
    throw new Error(err.error.message);
  }

  return res.json();
};

// 使用示例 / Usage
const result = await checkAddress("0x742d35Cc6634C0532925a3b844Bc9e7595f2bD18");
console.log(result.risk_level); // "critical"

Python (requests)

import requests

API_TOKEN = "your_access_token"
BASE_URL  = "https://api.threatledger.net/v1"

def check_address(address: str) -> dict:
    resp = requests.get(
        f"{BASE_URL}/addresses/check",
        params={"address": address},
        headers={"Authorization": f"Bearer {API_TOKEN}"},
    )
    resp.raise_for_status()
    return resp.json()

# 批量检查 / Batch check
def batch_check(addresses: list) -> dict:
    resp = requests.post(
        f"{BASE_URL}/addresses/batch",
        json={"addresses": addresses},
        headers={"Authorization": f"Bearer {API_TOKEN}"},
    )
    resp.raise_for_status()
    return resp.json()

# 使用示例 / Usage
result = check_address("0x742d35Cc6634C0532925a3b844Bc9e7595f2bD18")
print(f"Risk: {result['risk_level']} ({result['risk_score']}/100)")
# Risk: critical (87/100)
